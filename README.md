# QuantHub

Does what it says on the tin.

## Project setup

```shell
yarn install
```

### Compiles and hot-reloads for development

```shell
yarn run serve
```

### Compiles and minifies for production

```shell
yarn run build
```

### Lints and fixes files

```shell
yarn run lint
```

### Run your unit tests

```shell
yarn run test:unit
```

### Run your end-to-end tests

```shell
yarn run test:e2e
```
